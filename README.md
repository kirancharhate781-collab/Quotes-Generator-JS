# Quotes Generator JS

A simple and elegant web app built with HTML, CSS and JavaScript that displays random quotes.  
Every click generates a new quote for inspiration.

---

## Features

- Generate random quotes with a single click  
- Smooth UI update for each new quote  
- Clean, minimal design for easy reading  
- Works on both desktop and mobile
